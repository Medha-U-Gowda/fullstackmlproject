Required Columns

Your dataset must include all of the following columns, with exact names as shown.

age Patient age (numeric)
sex Gender — Male / Female
family_history Family history of genetic disease (1 = Yes, 0 = No)
snp_1 … snp_10 SNP genetic markers (values: 0, 1, or 2)
gene_expr_1 … gene_expr_5 Gene expression values (decimal numbers)
disease Output column — 1 = Has disease, 0 = No disease
💡

Formatting Rules

  • Use lowercase for all column names
  • Use underscores (_), not spaces
  • Output column (disease) must be the last column
  • File format must be .xlsx (Excel)
⚠️

Column names must match EXACTLY

  • Any mismatch may prevent the system from processing your dataset correctly.
